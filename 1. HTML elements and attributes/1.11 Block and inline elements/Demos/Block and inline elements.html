You are now able to identify elements as block or inline elements.

A block element
Begins on a new line. Usually space are added above and below the entire element.
An inline element
Does not start on a new line, but just goes with the flow.

A header is a block element

Square meters (<sup> is an inline element): The garage is m2 large.
