Important terms:

HTML
Hyper-text markup language (HTML) is a set of symbols inserted into files intended for display on the world wide web. The symbols tell web browsers how to display words and images and they direct the browser to link to other pages on the world wide web via hyperlinks.
URL
A URL (uniform resource locator) or web address is the string of characters you type into a browser to access a particular website or other resource on the Internet. (eg. http://www.baaa.dk )
JPEG
JPEG stands for Joint Photographic Experts Group, which was the committee that created the file format known as JPEG. The format is commonly used for photos displayed on the world wide web.
