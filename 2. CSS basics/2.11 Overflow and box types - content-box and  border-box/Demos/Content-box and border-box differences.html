Content-box and border-box differences

Content-box (default)

This is a contentbox

    .contentbox {
        box-sizing: content-box; /* This the the default value */
        border: 1px solid black;
        width: 125px;
        height: 100px;
        padding: 20px;
        margin: 10px;
    }
     --------------
     Content area width: 125px;
     Content area height: 100px;
     
     Total element width: 10 + 1 + 20 + 125 + 20 + 1 + 10 = 187px
     Total element height: 10 + 1 + 20 + 100 + 20 + 1 + 10 = 162px;

Border-box (preferred)

This is a borderbox

    .borderbox {}
        box-sizing: border-box;
        border: 1px solid black;
        width: 125px;
        height: 100px;
        padding: 20px;
        margin: 10px;
    }
     --------------
     Content area width: 125px - (2 * 20) - (2 * 1) = 83px
     Content area height: 100px - (2 * 20) - (2 * 1) = 58px
     
     Total element width: 10px + 125px + 10px = 145px 
     Total element height: 10px + 100px + 10px = 120px;
