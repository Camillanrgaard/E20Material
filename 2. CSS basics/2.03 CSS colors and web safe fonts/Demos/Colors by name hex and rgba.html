Colors by name, hex and rgba

140 color names are predefined in the HTML and CSS color specification. For example: blue, red, coral, brown, etc.

    color: red;
    background-color: black;

A hexadecimal color is specified with: #RRGGBB, where the RR (red), GG (green) and BB (blue) hexadecimal integers specify the components of the color. All values must be between 00 and FF.

    color: #ff0000;
    background-color: #000000;

RGBA color values are an extension of RGB color values with an alpha channel - which specifies the opacity of the object: rgba(red, green, blue, alpha). The alpha parameter is a number between 0.0 (fully transparent) and 1.0 (fully opaque).

    color: rgb(255, 0, 0, 0.7);
    background-color: rgba(0, 0, 0, 0.1);
