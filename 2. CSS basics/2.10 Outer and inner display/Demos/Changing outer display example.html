Changing outer display example

Example #1 - Changing an inline element to a block element

    HTML:
    <img src="images/tyre.jpg" alt="Image of a tyre" class="image__element">
    
    CSS: 
    .image__element {
        display: block;
        margin: 0 auto;  <-- This is how to center block elements
    }

Example #2 - Changing a block element to an inline element

Normal list:

  1. First
  2. Second
  3. Third

List with inline list elements:

  1. First
  2. Second
  3. Third
    HTML:
    <ol>
        <li class="listitem">First</li>
        <li class="listitem">Second</li>
        <li class="listitem">Third</li>
    </ol>
    
    CSS: 
    .listitem {
        display: inline;
    }
